# BEM FTIRS ITS Website

React + Vite + TypeScript website for **BEM FTIRS ITS**.

## Requirements

- Node.js 18+ (recommended)

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The production output is generated into `dist/`.

## Deployment

- **Vercel / Netlify**: deploy as a static site.
  - **Build command**: `npm run build`
  - **Output directory**: `dist`
